Study Data Collection

Important: Since Google Forms is blocking automated submissions, participants will download their responses as a CSV file after completing the study.

For Participants:

  1. Complete all 10 code comparison trials
  2. At the end, click "Download Responses (CSV)"
  3. Email the CSV file to the researcher

For Researchers:

Combining CSV files:

Save this Python script as combine_responses.py:

import os
import pandas as pd
from pathlib import Path

# Directory containing CSV files
csv_dir = Path("responses")
csv_files = list(csv_dir.glob("study_responses_*.csv"))

# Combine all CSV files
all_data = []
for file in csv_files:
    df = pd.read_csv(file)
    all_data.append(df)

# Create combined dataframe
combined_df = pd.concat(all_data, ignore_index=True)

# Save to single file
combined_df.to_csv("all_responses.csv", index=False)
print(f"Combined {len(csv_files)} files into all_responses.csv")
print(f"Total responses: {len(combined_df)}")
