üöÄ Testing the User Study Website

To test the website locally, you need to run a local server (required for loading JSON data).

Option 1: Using Python (Recommended)

Open terminal in the docs directory and run:

cd docs
python3 -m http.server 8000

Then open: http://localhost:8000

Option 2: Using Node.js

If you have Node.js installed:

npx http-server docs -p 8000

Then open: http://localhost:8000

Option 3: Using VS Code Live Server

If using VS Code with Live Server extension:

  1. Right-click on docs/index.html
  2. Select "Open with Live Server"

üîß Google Forms Integration

To connect with Google Forms:

  1. Create a Google Form with fields for:
    • Participant ID
    • Responses (JSON string)
    • Assigned Tuples
    • Completion Time
    • Demographics (age, gender, email, etc.)
  2. Get the form ID from the URL
  3. Get entry IDs by inspecting form elements
  4. Update js/study.js line ~240 with your form details
